# Microclimate

- Rasberry Pi
1. Rain Gauge : SDA SCL (I2C)
2. ENS160 : SDA SCL (I2C)
3. DS18B20 : GPIO 26

- Arduino Mega 2560
1. Pyranometer : RE 8, DE 7, RO 10 dan DI 11 --> Module Max485
2. Anemometer : RXD2 dan TXD2 (Serial 2) --> Module XY485
3. Wind Direction : RXD3 dan TXD3 (Serial 3) --> Module XY485
4. SHT85 : SDA SCL (I2C)
